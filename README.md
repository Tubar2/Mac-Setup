# Mac-Setup

My mac setup after a reboot or new buy

### Before anything

Search for Updates ( → Syst. Preferences → Software Update)

### Mac Preferences

Dock (⌘+space → Dock):
  - Remove apps
  - Change size
  - Hide
  - Clip to left/right side
  
Finder (Finder → ⌘+,): 
  - Open new finder windows on Home (General)
  - Edit Sidebar
  - Show Path Bar
  
 Trackpad: 
  - Point&Click → Tap to click
